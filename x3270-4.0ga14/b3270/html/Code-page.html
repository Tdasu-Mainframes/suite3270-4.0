code-page Indication

Indicates support for a particular host code page. Each indication may include one more more aliases with the attributes alias1, alias2, etc.

Appears only within a code-pages element.

AttributeAlways present?Purpose
nameyesCanonical code page name
alias1noFirst alias for the code page
alias2noSecond alias for the code page

Example: 

<code-pages> 
<code-page name="cp1140" alias1="us-euro"/> 
</code-pages>
