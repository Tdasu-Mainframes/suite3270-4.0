Name

mitm - network stream trace facility

Synopsis

mitm [-p listenport] [-f outfile]

Description

mitm is a proxy server that traces the data passing through it. It supports the Sun passthru protocol, where the client writes the desired host name and port, separated by a space and terminated by a carriage return and line feed, at the beginning of the session.

Network data is written in hexadecimal to the specified file.

The name is derived from its position in the network stream: the man in the middle.

Options

-p listenport
Specifies the port to listen on. The default port is 4200.
-f outfile
Specifies the trace file to create. The default is /tmp/mitm.pid.

Example

The emulator command-line option to route connection through mitm is:

-proxy passthru:127.0.0.1:4200
