attr Indication

Describes changed screen attributes over a region, with the text remaining the same. Always nested inside of a row element. If a particular screen attribute is not specified by this element, then it stays the same.

AttributeAlways present?Purpose
columnyesColumn number, with 1 at the left
countyesNumber of affected columns
fgnoForeground color
bgnoBackground color
grnoGraphic rendition
